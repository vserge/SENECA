Modelica.Fluid.Interfaces.PartialHeatTransfer

Единый интерфейс для моделей теплопередачи

Информация

Этот компонент - единый интерфейс для моделей теплопередачи. Уровни потока тепла Q_flows[n] через границы сегментов потока n получены как функция термодинамических состояний сегментов потока для данного жидкого носителя, surfaceAreas[n] и граничных температур heatPorts [n].T.

Коэффициент тепловых потерь k может использоваться, чтобы смоделировать тепловую изоляцию между heatPorts. T и T_ambient (температура внешней среды).

Расширяющая модель, которая реализует данный интерфейс должна определить одно уравнение: отношение между предопределенными температурами жидкости Ts[n], граничными температурами heatPorts [n].T и уровнем потока тепла Q_flows[n].
